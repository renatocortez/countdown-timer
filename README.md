# countdown-timer
 countdown-timer
 
In this tutorial, you will learn how to make a countdown clock which can be used when a new product is coming out or a sale is about to end on an ecommerce site.

Key concepts covered:

getFullYear()

getMonth()

getDate()

Math.floor()

setInterval()

clearInterval()
